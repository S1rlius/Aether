##  Что по безопасности?

Я не стал изобретать свои алгоритмы, а взял проверенные инструменты:
* **Шифрование:** XSalsa20-Poly1305.
* **Обмен ключами:** Curve25519 (Diffie-Hellman).
* **Forward Secrecy:** Ключи живут только в рамках одной сессии. Если кто-то украдет ваш лог завтра, он не вскроет то, что вы писали сегодня.
* **Аутентификация (TOFU):** Первая проверка ключа делается вручную, потом клиент запоминает собеседника в `aether_contacts.conf`. Если ключ подменят — программа заорет.

##  Как это запустить?

### Требования
Нужен Linux (тестил на Arch), установленный `libsodium` и `gcc`.

### Сборка
Я специально разделил сервер и клиент, чтобы их можно было разносить по разным машинам.

```bash
# Собираем серверную часть
gcc server.c -o aether-server -lpthread

# Собираем клиента
gcc client.c -o aether-client -lsodium -lpthread
